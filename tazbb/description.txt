
SliTaz Build Bot. Tazbb is a tool to automate and test SliTaz package 
building. Please read the Cookbook documentation for more information 
and put your ideas on the Tazbb Wiki page on the Labs. The package 
provides the tazbb script, a config file and a web interface.

Tank Build Bot : http://bb.slitaz.org/
Tazbb Wiki     : http://labs.slitaz.org/wiki/distro/Tazbb
Cookbook page  : http://doc.slitaz.org/en:cookbook:buildbot
