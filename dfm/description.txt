Dino - Simple File Manager for GNU/Linux.

Features
 * Easy to use
 * Tabs support
 * Multi selection
 * Customizable interface
 * Drag & drop support
 * Supported file operations: New dir/file, open, run, delete, rename, cut,
   copy, paste, edit.
 * Thumbnail view
 * Built-in text editor
 * Custom actions
 * List and tree view
