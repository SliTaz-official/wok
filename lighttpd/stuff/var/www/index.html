English

Welcome to free LightTPD web server index page, powered by SliTaz GNU/Linux operating system. You can delete or modify this page to put your own xHTML pages, web sites, CGI or PHP scripts.

Configuration file can be found in /etc/lighttpd/lighttpd.conf and is autodocumemted. A page in the SliTaz Handbook is dedicated to the web server configuration. The official documentation can be found at http://trac.lighttpd.net/trac/wiki#Documentation.

Français
